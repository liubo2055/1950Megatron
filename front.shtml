Welcome to COMP 1950!

It’s September 2014, HTML5, CSS3 and JS are generally well accepted, Microsoft has made a half decent browser and front-end web development is a solid job on it’s own!

In COMP 1950, we’re going to take a look at that front end stack, how it works and how you can build pages with those three technologies.

During the first half of the course, we will take a hard look at HTML and CSS and where we are now as web developers. We will move on to Template development (the primary task in web development these days) and sprinkle on some JS later on.

This page will contain news or updates from time to time, so stay tuned!

Let’s get started.
